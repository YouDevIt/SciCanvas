Sci·Canvas
LIVE SCIENTIFIC GRAPHICS ENGINE — MANUALE COMPLETO

SciCanvas è un'applicazione web autocontenuta (singolo file .html) per la creazione e visualizzazione interattiva di grafica scientifica nel browser. Integra un editor di codice JavaScript con API grafica, un simulatore fisico 2D completo e un costruttore di geometria euclidea dinamica — senza richiedere installazioni o dipendenze esterne.

⚡ Zero installazione 🖥 Single-file HTML 📐 3 moduli integrati 🔢 CAS simbolico 📄 GNU GPL v3.0

01 Presentazione

SciCanvas è progettato per studenti, docenti, ricercatori e appassionati di matematica e fisica che vogliono realizzare visualizzazioni scientifiche di qualità direttamente nel browser. Basta aprire SciCanvas.html in un browser moderno.

⌨

CODICE

Editor JavaScript con API grafica Canvas, REPL JS + CAS, libreria esempi, documentazione API integrata e gestore script personali.

⛛

SIM

Simulatore fisico 2D con corpi rigidi, vincoli cinematici, forze globali configurabili anche come espressioni dinamiche, integratore RK4 e CustomDraw.

△

GEO

Costruttore di geometria euclidea dinamica: spostando un punto le figure dipendenti si aggiornano in tempo reale. Include slider parametrici, vista algebrica e scie.

Librerie Esterne Integrate

Rendering di formule LaTeX in SVG direttamente sul canvas HTML5.

03 Modulo CODICE

Il modulo CODICE permette di scrivere ed eseguire JavaScript che disegna sul canvas tramite un'API grafica di alto livello. È organizzato in cinque sotto-schede.

3.1 Editor

La sotto-scheda 📝 editor è l'ambiente principale di scrittura.

  • Syntax highlighting in tempo reale — parole chiave, numeri, stringhe e commenti colorati tramite un layer sovrapposto all'area di testo, senza latenze percettibili.
  • Numerazione righe — colonna a sinistra sincronizzata con lo scroll in ogni momento.
  • Indentazione con Tab — inserisce spazi senza perdere il focus.
  • Drag & drop file — trascinare un file .js o .txt sull'editor ne carica il contenuto. Un overlay "⬇ Rilascia il file .js / .txt" appare durante il trascinamento.
  • Esecuzione — pulsante ▶ ESEGUI o scorciatoia Ctrl+↵.
  • Stop animazione — ■ STOP interrompe il loop requestAnimationFrame impostando animRun = false.
  • Clear canvas — ⊘ CLEAR cancella il canvas senza fermare eventuali animazioni.
  • Ridisegno automatico al resize — se non è attivo un loop di animazione, la funzione redrawStatic() viene chiamata automaticamente al ridimensionamento della finestra.

3.2 REPL Interattivo

La sotto-scheda ⚡ repl offre un Read-Eval-Print Loop per eseguire espressioni singole istantaneamente.

ModalitàPulsanteDescrizione
JSJS Valuta espressioni JavaScript nell'ambiente del canvas. Testa funzioni API, ispeziona variabili, invoca comandi grafici istantanei.
CASCAS Valuta espressioni di algebra simbolica via Algebrite: derivate, integrali, semplificazione, fattorizzazione, espansione, risoluzione simbolica.

Funzionalità UI del REPL

  • Namespace _ — variabili assegnate come _.nome = valore persistono tra una valutazione e l'altra. La barra replNsBar mostra il contenuto corrente di _.
  • Area di output — risultati con formattazione distinta per valori, errori e output CAS.
  • Cronologia — ↑/↓ navigano la cronologia della sessione.
  • ⊘ Clear — pulisce l'output senza azzerare il namespace.
  • ? Aiuto — guida sintetica ai comandi e all'API disponibile nel REPL.

Esempi in modalità CAS

// Derivata simbolica
derivative(sin(x), x)          → cos(x)

// Semplificazione
simplify((x^2 - 1) / (x - 1)) → x + 1

// Integrale indefinito
integral(x^2, x)               → x^3 / 3

// Fattorizzazione
factor(x^3 - x^2 - x + 1)    → (x - 1)^2 * (x + 1)

// Espansione
expand((a + b)^3)               → a^3 + 3*a^2*b + 3*a*b^2 + b^3

Esempi in modalità JS

drawAxis()
plotFunction(x => Math.sin(x))
_.f = x => x*x - 2*x + 1
plotFunction(_.f, -2, 4)

3.3 Libreria Esempi

La sotto-scheda 📚 esempi contiene script dimostrativi pre-caricati per categoria. Un click carica il codice nell'editor. Coprono tipicamente:

  • Grafici di funzioni matematiche (trigonometriche, esponenziali, logaritmiche, polinomiali).
  • Curve parametriche, spirali di Archimede, curve di Lissajous, ipotrocoidi.
  • Animazioni in loop con requestAnimationFrame.
  • Formule LaTeX come etichette tipograficamente corrette sul canvas (MathJax).
  • Visualizzazioni di fisica: campi vettoriali, traiettorie, distribuzioni.
  • Grafici di dati, istogrammi, scatter plot, bar chart.
  • Pannelli di controllo parametrici con lil-gui.

04 Modulo SIM — Simulatore Fisico 2D

Il modulo ⛛ SIM è un simulatore fisico bidimensionale completo. Permette di costruire scene con corpi rigidi, vincoli meccanici e forze globali, e di osservarne l'evoluzione tramite integrazione numerica RK4.

4.1 Corpi Fisici

Si aggiungono selezionando lo strumento dalla palette Corpi e cliccando sul canvas. Le proprietà fisiche sono configurabili nel Pannello Proprietà, visibile quando il corpo è selezionato.

StrumentoIconaDescrizione fisica
Selezione↖Seleziona corpi esistenti e trascinali per riposizionarli. Non aggiunge nuovi oggetti.
Particella●Massa puntiforme (I = 0). Ideale per proiettili, orbite, moto browniano, particelle in campo EM.
Blocco⮬Corpo rettangolare rigido con massa distribuita uniformemente. Reagisce a forze, coppie e collisioni con rotazione.
Sfera◉Corpo circolare, inerzia di sfera piena: I = ²⁄₅ mr². Collisioni elastiche e anelastiche configurabili.
Cilindro⬤Corpo circolare, inerzia di cilindro pieno: I = ½ mr². Utile per rulli e dischi rotolanti.
Asta━Corpo allungato rigido, inerzia di asta: I = ¹⁄₁₂ mL². Fondamentale per pendoli, leve, bracci articolati.
Anello○Corpo circolare cavo, inerzia di anello: I = mr². Utile per confronti dinamici con sfera e cilindro.

Proprietà configurabili (pannello Properties)

  • Massa m (kg) e dimensioni geometriche
  • Posizione iniziale x, y (m) e orientamento iniziale (rad)
  • Velocità lineare iniziale vx, vy (m/s) e angolare (rad/s)
  • Coefficiente di restituzione (0 = anelastico, 1 = elastico)
  • Coefficiente di attrito e colore/stile visivo

4.2 Vincoli e Superfici

Definiscono relazioni cinematiche tra corpi o tra corpo e ambiente. Si aggiungono dalla palette Vincoli.

StrumentoIconaDescrizione
Punto fisso📌Ancora un punto del corpo a una posizione fissa: permette la rotazione ma impedisce la traslazione. Essenziale per pendoli, ruote, strutture ancorate.
Perno🔘Collega due corpi con un perno rotante (cerniera): trasmette forze ma permette rotazione reciproca libera.
Distanza↔Mantiene costante la distanza tra due punti: si comporta come un'asta rigida inestensibile o un cavo teso.
Cerniera🔗Variante del perno con angolo di riposo e rigidità torsionale configurabili (snodo elastico).
Guida H⟶Il corpo scorre solo lungo una guida orizzontale (solo X). Simula binari, pistoni, carrelli orizzontali.
Guida V⟵Come Guida H ma in direzione verticale (solo Y).
Guida Ang⟳Guida orientata a un angolo arbitrario: il corpo scorre solo lungo la direzione specificata.
Piano⬜Piano di contatto orizzontale (pavimento/soffitto) con collisioni fisiche e attrito.
Inclinato╱Piano inclinato a un angolo configurabile. Simula rampe, scivoli, superfici oblique.
Fune〜Vincolo di lunghezza massima (solo trazione): passivo quando allentato, si tende quando i punti raggiungerebbero il limite di distanza.

4.3 Forze Globali

Campi di forza che agiscono su tutti i corpi della scena simultaneamente. Ogni campo si attiva/disattiva con una checkbox. Il valore può essere una costante numerica o un'espressione dinamica che dipende da x, y (posizione del corpo in m) e t (tempo simulato in s).

CampoSimboloUnitàDescrizione
Gravitàgm/s²Accelerazione gravitazionale verso il basso. Default: 9.81. Può essere un'espressione come sin(t) per gravità oscillante.
Campo EEx, EyN/CCampo elettrico nelle due componenti. Supporta espressioni spaziali (es. sin(y), -0.5*x).
Campo BBzTCampo magnetico perpendicolare al piano. Genera la forza di Lorentz sulle particelle in moto. Può dipendere da t.
Attrito viscosobN·s/mForza proporzionale e opposta alla velocità (smorzamento lineare). Simula resistenza dell'aria o fluido viscoso.

Variabili nelle espressioni

VariabileSignificato
x, yPosizione attuale del corpo in metri
tTempo fisico simulato corrente in secondi
Funzioni disponibili: sin cos tan sqrt exp log abs pow PI

Esempi di espressioni avanzate

g  = 9.81 + 2*sin(t)   // gravità oscillante nel tempo
Ex = -0.5*x             // forza elastica verso l'origine
Bz = 1.5               // campo magnetico costante
b  = 0.1*exp(-t)       // attrito che decade esponenzialmente

4.4 Integratore Numerico

ParametroDettaglio
dt — passo temporaleIntervallo in secondi. Range: 0.0001–0.05 s, default 0.005 s. Valori piccoli = più precisione ma più lento; valori grandi = più veloce ma potenzialmente instabile.
Metodo: RK4Runge-Kutta al 4° ordine: errore locale O(dt⁵), molto più stabile del metodo di Eulero per sistemi fisici realistici.
Mostra v (checkbox)Sovrappone al canvas i vettori velocità di ogni corpo (frecce proporzionali a modulo e direzione). Utile per analizzare cinematica e conservazione del momento.
Suggerimento: per simulazioni stabili con vincoli rigidi e alta rigidità, usare dt piccoli (0.001–0.002 s). Per simulazioni qualitative a lungo termine, è possibile aumentare il dt e usare il moltiplicatore vel× per accelerare.

05 Modulo GEO — Geometria Euclidea Interattiva

Il modulo △ GEO è un costruttore di geometria euclidea dinamica. Il principio fondamentale è il ridisegno dinamico: spostando un punto libero, tutte le figure dipendenti si aggiornano istantaneamente. Permette di esplorare proprietà geometriche e verificare congetture in modo visivo e interattivo.

5.1 Strumenti di Costruzione

Tutti gli strumenti si trovano nella palette Strumenti del pannello laterale.

5.2 Vista Algebrica

Il pannello Vista Algebrica (geo-alg-list) mostra la rappresentazione algebrica di tutti gli oggetti, aggiornata in real-time al trascinamento:

  • Punti — coordinate (x, y)
  • Segmenti — lunghezza
  • Rette — equazione
  • Cerchi — centro e raggio
  • Angoli — valore in gradi o radianti
  • Distanze — valore numerico
  • Slider — nome del parametro e valore corrente

Supporta anche l'input diretto di espressioni: è possibile digitare coordinate o definizioni analitiche per creare oggetti senza usare gli strumenti grafici.

Quando non ci sono oggetti: "Nessun oggetto. Usa gli strumenti o digita un'espressione."

5.3 Nota Costruzione

Il campo ✎ Nota costruzione è una textarea in cui inserire un commento o una spiegazione testuale. Il testo viene visualizzato direttamente sul canvas come annotazione ed è incluso nell'esportazione HTML standalone. Utile per materiali didattici, presentazioni e condivisione della costruzione con annotazioni esplicative.

5.4 Griglia e Scie

FunzioneIconaDescrizione
Griglia⊞ Attiva/disattiva una griglia di riferimento sul canvas. Facilita la lettura delle coordinate e la costruzione di figure con misure precise.
Scie⌁ Attiva/disattiva la modalità traccia: quando attiva, i punti dipendenti lasciano una scia grafica durante il trascinamento di un punto libero o lo spostamento di uno slider. Permette di visualizzare il luogo geometrico descritto da un punto al variare della costruzione — strumento potente per l'esplorazione geometrica.
Esempio d'uso delle Scie: costruisci il punto medio di un segmento i cui estremi si muovono su due cerchi. Attiva Scie e trascina uno degli estremi: vedrai apparire il luogo geometrico del punto medio, che risulterà essere a sua volta un cerchio.

5.5 Pannello Proprietà

Il pannello Proprietà appare selezionando un oggetto. Consente di modificare:

  • Nome/etichetta dell'oggetto (rinominabile).
  • Colore e stile di visualizzazione: spessore linea, tratteggio, riempimento.
  • Valori numerici: coordinate (punti), raggio (cerchi), valore e intervallo (slider).
  • Visibilità: mostrare/nascondere l'etichetta o l'oggetto stesso.
  • Informazioni sui vincoli: da quali oggetti dipende e quali oggetti dipendono da esso.
